# 🔄 Calculator Functionalities

## ➕ Arithmetic Operations

The calculator supports:

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)

---

## 🧠 Advanced Functionalities

Users can also use:

- Square Root (√)
- Decimal Numbers (.)
- Plus / Minus Toggle (±)
- Brackets ()
- Answer Recall (ANS)

---

## ⚙️ Utility Buttons

### 🗑️ Delete Button

Deletes the last entered character.

Example:

```bash
123+
```

After clicking DEL:

```bash
123
```

---

### 🧹 Clear Button

Clears:

- Expression Screen
- Result Screen

---

### ✅ Enter Button

Evaluates the mathematical expression and displays the result.
