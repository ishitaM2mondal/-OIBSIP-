# 🔑 Authentication Flow

## 📝 Registration

Users can create an account using:

- Username
- Password

The credentials are stored using:

```javascript
localStorage.setItem()
```

---

## 🔓 Login

The system checks user credentials with stored data.

```javascript
if(username === storedUsername && password === storedPassword)
```

If valid:

✅ Login Successful  
✅ Dashboard Access Granted

---

## 🚪 Logout

Users can securely logout and return to the login screen.
